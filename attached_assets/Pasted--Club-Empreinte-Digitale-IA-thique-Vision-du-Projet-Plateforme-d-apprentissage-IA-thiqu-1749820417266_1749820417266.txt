# Club Empreinte Digitale - IA Éthique

## 🌍 Vision du Projet

Plateforme d'apprentissage IA éthique avec écosystème solidaire intégré, combinant formation responsable, commerce équitable et accessibilité universelle.

## 🎯 Mission

Démocratiser l'accès à l'éducation technologique tout en promouvant une approche éthique de l'intelligence artificielle et en soutenant l'économie circulaire à travers notre boutique solidaire Costa del Sol.

---

## 🏗️ Architecture du Projet

### Frontend (React + TypeScript)
- **Framework** : React 18 avec TypeScript
- **Routing** : Wouter
- **UI** : Shadcn/ui + Tailwind CSS
- **État** : React Query pour la gestion des données
- **Formulaires** : React Hook Form + Zod

### Backend (Express + Node.js)
- **Serveur** : Express.js
- **Base de données** : PostgreSQL avec Drizzle ORM
- **Session** : Express Session
- **IA** : OpenAI API pour Chat IARP

## 📦 Installation et Déploiement

### Prérequis
```bash
Node.js 18+
PostgreSQL 14+
npm ou yarn
```

### Installation Locale
```bash
# Cloner le projet
git clone [repository-url]
cd club-empreinte-digitale

# Installer les dépendances
npm install

# Configuration environnement
cp .env.example .env
# Configurer DATABASE_URL et OPENAI_API_KEY

# Migration base de données
npm run db:migrate

# Démarrer le serveur de développement
npm run dev
```

### Structure des Fichiers
```
├── client/src/
│   ├── components/          # Composants React
│   ├── pages/              # Pages principales
│   ├── hooks/              # Hooks personnalisés
│   └── lib/                # Utilitaires
├── server/
│   ├── routes.ts           # API Routes
│   ├── storage.ts          # Couche données
│   └── openai.ts           # Intégration IA
├── shared/
│   └── schema.ts           # Schémas base de données
└── package.json
```
