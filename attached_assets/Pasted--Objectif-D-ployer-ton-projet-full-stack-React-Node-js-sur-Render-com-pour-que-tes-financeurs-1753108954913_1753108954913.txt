⚙️ PRÉREQUIS TECHNIQUES

Tu dois avoir :
	•	Un compte Render (gratuit)
	•	Ton projet sur GitHub (public ou privé avec token)
	•	Un frontend (React/Vite) dans client/
	•	Un backend (Node/Express) dans server/

⸻

🧱 STRUCTURE DE BASE (tu l’as déjà 👏)
bash : copier/coller :                      club-empreinte-digitale/
├── client/      → React (Vite)
├── server/      → Node.js / Express
├── shared/      → (OK)
├── .env         → à configurer pour Render


🚀 1. DÉPLOYER LE BACKEND (Express.js)
	1.	Connecte-toi à Render
	2.	Clique “New” > “Web Service”
	3.	Choisis ton repo GitHub
	4.	Configure :
	•	Name: ced-api-backend
	•	Root directory: server/
	•	Runtime: Node
	•	Build Command: npm install
	•	Start Command: node index.js ou npm run start selon ton script
	•	Environment: ajoute les variables .env nécessaires :
	•	PORT=10000
	•	MONGO_URI=... (ou autre DB)
	5.	Clique “Create Web Service”

✅ Ton API est maintenant hébergée.

⸻

🌍 2. DÉPLOYER LE FRONTEND (React / Vite)
	1.	Retourne sur Render
	2.	Clique “New” > “Static Site”
	3.	Choisis ton repo GitHub
	4.	Configure :
	•	Name: ced-ui-frontend
	•	Root directory: client/
	•	Build Command: npm install && npm run build
	•	Publish Directory: dist
	•	Environment:
	•	VITE_API_URL=https://ced-api-backend.onrender.com
	5.	Clique “Create Static Site”

✅ Ton UI est maintenant déployée publiquement avec React/Vite.

⸻

📌 3. PERSONNALISER TON DOMAINE

Dans chaque service Render :
	•	Clique sur “Settings”
	•	Ajoute un custom domain comme cedbank.org ou prettyhowq.org
	•	Ou utilise l’URL gratuite https://ced-ui-frontend.onrender.com
